コンピュータ環境構築(macOS10.14以降)

*Windowsユーザの方はこちら

  • 以下の7つのツールをインストールする.
  1. GNU Fortranコンパイラ
  2. Java14
  3. 地震波形解析ソフト(Seismic Analysis Code)
  4. 地図描画ソフト(Generic Mapping Tools)
  5. 走時計算ソフトウェア(TauP)
  6. 理論地震波形計算ソフトウェア(DSM)
  7. 地震波形解析ツール(Kibrary)
  1. GNU Fortran コンパイラのインストール
  • ターミナルを開いて以下のコマンドをコピ&ペーストしてリターンキーを押して実行する.
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    brew install gcc
    ターミナルで以下のコマンドを実行してGNU Fortranコンパイラのバージョンが表示されればOK.
    gfortran –-version
  • 以下のコマンドでopen-mpiをインストールする.
    brew install open-mpi
    ターミナルで以下のコマンドを実行してopen-mpiのバージョンが表示されればOK.
    mpif90 –-version
  1. Java14のインストール
  • Java14を以下のコマンドを実行してインストール.
    brew tap AdoptOpenJDK/openjdk
    brew install adoptopenjdk14 --cask
  • java_home コマンドでインストールされたディレクトリの場所を確認する.
    /usr/libexec/java_home -v 14
    うまくいけば下記のように表示される.
/Library/Java/JavaVirtualMachines/adoptopenjdk-14.jdk/Contents/Home
  • 環境変数 JAVA_HOME と PATH を設定する.
    echo export JAVA_HOME=/Library/Java/JavaVirtualMachines/adoptopenjdk-14.jdk/Contents/Home >> ~/.zshrc
    echo PATH=${JAVA_HOME}/bin:${PATH} >> ~/.zshrc
    以下のコマンドで正しく追記されているかを確認する.
    tail ~/.zshrc
    問題なければ以下のコマンドで反映.
    source ~/.zshrc
  • インストールされた AdoptOpenJDK を確認する.
    java -version
    以下のように表示されていればOK.
openjdk version "14" 2020-03-17

OpenJDK Runtime Environment AdoptOpenJDK (build 14+36)

OpenJDK 64-Bit Server VM AdoptOpenJDK (build 14+36, mixed mode, sharing)
  1. 地震波形解析ソフト(SAC)のインストール
  • 下記のURLから圧縮ファイルをダウンロード
    https://bit.ly/3tBVA4V
  • 展開すると「sac」というディレクトリができるので、以下のコマンドを実行する.
    sudo mv ~/Downloads/sac /usr/local
  • 以下のコマンドを実行して環境変数を設定する.
    echo export SACHOME=/usr/local/sac >> ~/.zshrc
    echo export SACAUX=/usr/local/sac/aux >> ~/.zshrc
    echo export PATH=/usr/local/sac/bin:\$PATH >> ~/.zshrc
    以下のコマンドで変更を反映.
    source ~/.zshrc
  • 以下のコマンドでXquartzをインストール
    brew install xquartz --cask
  • ターミナル上でsacと入力してSACのプロンプトが表示されればOK.
  • 以下のコマンドでテストデータをプロットする.
    curl -O https://bit.ly/3ezSV5s
    sac
    $sac\> read gaussian8s.sac
    $sac\> p1 ガウス関数が表示されればOK.
  1. 地図描画ソフト(GMT)のインストール
  • M1 Macの場合は以下のコマンドで別途x86版homebrewをインストールする.
    arch -x86_64　/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    arch -x86_64 /usr/local/Homebrew/bin/brew install gmt
    brew install ghostscript graphicsmagick ffmpeg
  • Intel CPU搭載のMacの場合は、以下のコマンドを実行してGMTと関連するライブラリをインストール.
    brew install gmt ghostscript graphicsmagick ffmpeg
  • 以下のテストを実行して正しくインストールされているかを確認する.
    以下のリンクからスクリプトファイルをダウンロード.
    https://bit.ly/32AEad4
    以下のコマンドを実行して日本地図が表示されればOK.
    sh checkGMT.sh
  • 問題がなければ、日本地図が描画される.
  1. 走時計算ソフトウェア(TauP)のインストール
  • 以下のコマンドでインストール.
    curl -O http://www.seis.sc.edu/downloads/TauP/TauP-2.5.0.tgz && tar zxf TauP-2.5.0.tgz
    • sudo mv TauP-2.5.0 /usr/local
  • 環境変数を設定する.
    echo export TAUP_HOME=/usr/local/TauP-2.5.0 >> ~/.zshrc
    echo export PATH=${TAUP_HOME}/bin:\$PATH >> ~/.zshrc
    以下のコマンドで正しく追記されているかを確認する.
    tail ~/.zshrc
    問題なければ以下のコマンドで反映.
    source ~/.zshrc
  • 以下のコマンドを実行して確認.
    taup_time -mod prem -h 400 -deg 80 -ph P,PcP,S,ScS
    問題なければ以下のように表示される.
Model: prem

以下のコマンドを実行するとGUIでtaup_time, taup-pierce, taup_pathを実行できる.
taup

  1. 理論地震波形計算ソフトウェア(DSM)のインストール
  • 下記のリンクから圧縮ファイルをダウンロードして解凍する.
    https://bit.ly/2Qnrq74
  • 以下のコマンドを実行してビルドする.
    make tipsv
    *いくつかのWarning(警告)が出るが問題ない.
  • 以下のコマンドを実行してテスト計算を行い、ファイルが作成されることを確認する. 作成されたファイルはexample/にABC.20090101psv.spc及びXYZ.20090101psv.spcとして格納されている.
    ./tipsv < example/tet1.inf
  1. 波形解析ツール(Kibrary)のインストール
  • 以下のコマンドでインストール.
    kins=$(mktemp) && curl -sL -o $kins https://bit.ly/3xa6DVb && /bin/sh $kins && rm -f $kins
    もしくは、
    kins=$(mktemp) && wget -q -O $kins https://bit.ly/3xa6DVb && /bin/sh $kins && rm -f $kins
  • 以下のコマンドで問題なくインストールされているかを確認.
    ~/Kibrary//bin/kibrary_operation

コンピュータ環境構築(Windows10)

  • 以下の7つのツールをインストールする.
  1. WSL (Windows Subsystem for Linux)
  2. GNU Fortranコンパイラ
  3. Java14
  4. 地図描画ソフト(Generic Mapping Tools)
  5. 地震波形解析ソフト(Seismic Analysis Code)
  6. 走時計算ソフトウェア(TauP)
  7. 理論地震波形計算ソフトウェア(DSM)
  8. 地震波形解析ツール(Kibrary)
  1. WSL (Windows Subsystem for Linux)をインストールする.
  • Windows上でLinuxプログラムを動作できる仮想環境.
  • GUI、またはPowerShellを使って、Windowsのオプション機能「Windows Subsystem for Linux」を有効にする. GUIであれば、スタート画面や設定アプリで「Windowsの機能1」まで入力して検索するか、コントロールパネルから「コントロールパネル\プログラム\Windowsの機能の有効化または無効化」と辿る.



    Windows Subsystem for Linux（Linux 用 Windows サブシステム）」と「仮想マシン プラットフォーム」を有効にする. 再起動を求められるので、再起動する.
  • こちらからカーネル更新プログラムをダンロードして、開いてインストール.
  • Linuxディストリビューションをインストールする. Microsoft Storeアプリを起動して「WSL」を検索する. 表示されている「Ubuntu 20.04LTS」を選択しインストール.
  • Ubuntuをインストールした段階ではまだ有効化されていないので、初期化して有効にする必要がある. そのために、ストアアプリから起動ボタンを押すか、もしくはスタート画面からディストリビューションを選択.


    ユーザー名、パスワードを入力し初期化完了.
  1. GNU Fortranコンパイラのインストール
  • 以下のコマンドでパッケージインデックスを更新.
    sudo apt update && sudo apt -yV upgrade
    パスワードを聞かれた場合は先ほど設定したパスワードを入力.
  • 以下のコマンドでgfortranをインストール
    sudo apt install gfortran-10 make gdb build-essential
    sudo ln -s `which gfortran-10` /usr/bin/gfortran
  • 以下のコマンドで正常にインストールされていることを確認.
    gfortran --version
    問題なければ以下のように表示される.
GNU Fortran (Homebrew GCC 10.2.0_4) 10.2.0

  1. Java14のインストール

  • 以下のコマンドを1行ずつ実行してjava14をインストール
    sudo apt install openjdk-14-jdk

  • 環境変数 JAVA_HOME と PATH を設定する.
    export JAVA_HOME=”/usr/lib/jvm/java-14-openjdk-amd64” >> ~/.bashrc
    echo PATH=${JAVA_HOME}/bin:${PATH} >> ~/.zshrc
    以下のコマンドで正しく追記されているかを確認する.
    tail ~/.zshrc
    問題なければ以下のコマンドで反映.
    source ~/.zshrc

  • 以下のコマンドを実行してインストールを確認.
    java --version

  1. 地図描画ソフト(GMT)のインストール
  • 以下のコマンドを実行してGMTをインストールする.
    sudo apt-get install gmt gmt-dcw gmt-gshhg
    sudo apt-get install ghostscript
    sudo apt-get install gdal-bin graphicsmagick ffmpeg
    • 以下のテストを実行して正しくインストールされているかを確認する.
      以下のリンクからスクリプトファイルをダウンロード.
      https://bit.ly/32AEad4
      以下のコマンドを実行して日本地図が表示されればOK.
      sh checkGMT.sh
  • 問題がなければ、日本地図が描画される.
  1. 地震波形解析ソフト(SAC)のインストール
  • 下記のURLから圧縮ファイルをダウンロード
    https://bit.ly/2PdZeTG
  • 展開すると「sac」というディレクトリができるので、以下のコマンドを実行する.
    tar xvf /mnt/c/Users/*"username"*/Downloads/sac-102.
    sudo mv ./sac /usr/local
  • 以下のコマンドを実行して環境変数を設定する.
    echo export SACHOME=/usr/local/sac >> ~/.bashrc
    echo export SACAUX=/usr/local/sac/aux >> ~/.bashrc
    echo export PATH=/usr/local/sac/bin:\$PATH >> ~/.bashrc
    以下のコマンドで変更を反映.
    source ~/.bashrc
  • 以下のリンクからWindows用のXwindowシステムをダウンロード.
    https://bit.ly/3tOQAKk
    vcxsrv*.exeを管理者として実行(右クリックから).
    VcXsrvを選択して「次へ」.
    Download先はデフォルトのまま「インストール」.
  • xlaunch.exe実行ファイルを起動.
  • multiple window を選んで次へ、start no client を選んで次へ、Native openglのチェックを外し、Additional parameters for VcXsrvに"-ac"を追加して「次へ」.
  • Save configuration を押してデスクトプに保存し、完了を押してxlaunchを終了.
  • Windowsキー + rを押して, shell:startupと入力して検索し、開いたファイルブラウザ(スタートアップ)にデスクトップに保存したファイルをドラッグ＆ドロップで移動.
  • 以下のコマンドでbashrcに設定を追加.
    echo export DISPLAY=$(cat /etc/resolv.conf | grep nameserver | awk '{print $2}'):0 >> ~/.bashrc
    source ~/.bashrc
  • ターミナル上でsacと入力してSACのプロンプトが表示されればOK.
  1. 走時計算ソフトウェア(TauP)のインストール
  • 以下のコマンドでインストール.
    wget http://www.seis.sc.edu/downloads/TauP/TauP-2.5.0.tgz && tar zxf TauP-2.5.0.tgz
    sudo mv TauP-2.5.0 /usr/local
  • 環境変数を設定する.
    echo export TAUP_HOME=/usr/local/TauP-2.5.0 >> ~/.bashrc
    echo export PATH=${TAUP_HOME}/bin:\$PATH >> ~/.bashrc
    以下のコマンドで正しく追記されているかを確認する.
    tail ~/.zshrc
    問題なければ以下のコマンドで反映.
    source ~/.zshrc
  • 以下のコマンドを実行して確認.
    taup_time -mod prem -h 400 -deg 80 -ph P,PcP,S,ScS
    問題なければ以下のように表示される.

  1. 理論地震波形計算ソフトウェア(DSM)のインストール
  • 下記のリンクから圧縮ファイルをダウンロードして解凍する.
    https://bit.ly/2Qnrq74
  • 以下のコマンドを実行してビルドする.
    make tipsv
    *いくつかのWarning(警告)が出るが問題ない.
  • 以下のコマンドを実行してテスト計算を行い、ファイルが作成されることを確認する. 作成されたファイルはexample/にABC.20090101psv.spc及びXYZ.20090101psv.spcとして格納されている.
    ./tipsv < example/tet1.inf
  1. 波形解析ツール(Kibrary)のインストール
  • 以下のコマンドでインストール.
    kins=$(mktemp) && curl -sL -o $kins https://bit.ly/3xa6DVb && /bin/sh $kins && rm -f $kins

    kins=$(mktemp) && wget -q -O $kins https://bit.ly/3xa6DVb && /bin/sh $kins && rm -f $kins

    ~/Kibrary//bin/kibrary_operation
